# Завершение игры и статусы

## Статусы игры (`domain.GameStatus`)

- `waiting` – ожидание второго игрока (только PvP, после создания).
- `playing` – игра в процессе.
- `finished` – игра завершена (победа или ничья).

## Как определяется завершение?

- После каждого хода вызывается `logic.GameFinished(newGame)`.
- Функция проверяет 8 выигрышных линий и заполненность поля.
- Если игра закончена:
  - `newGame.Status = StatusFinished`
  - `newGame.WinnerID` устанавливается (ID победителя или `uuid.Nil` при ничьей).

## Использование в API

- При получении игры клиент видит статус и `winner_id`.
- Завершённые игры доступны через `GET /games/finished` (только для текущего пользователя).
