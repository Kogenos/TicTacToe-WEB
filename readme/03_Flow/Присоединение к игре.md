# Присоединение к игре (PvP)

1. Клиент (второй игрок) получает список доступных игр: `GET /games/available`.
2. Выбирает игру по UUID.
3. Отправляет `POST /game/{id}/join` с заголовком `Authorization`.
4. `GameHandler.JoinGame` → `GameServiceWithRepo.JoinGame`:
   - Загружает игру по UUID.
   - Проверяет, что статус `waiting`, `Bot == false` и `PlayerO_ID == nil`.
   - Назначает `PlayerO_ID` текущему пользователю.
   - Меняет статус на `playing`.
   - Сохраняет игру.
5. Возвращает обновлённый `GameDTO`.

Теперь оба игрока могут делать ходы через `POST /game/{id}/move`.

См. также [Создание игры (с ботом или PvP)](Создание игры (с ботом или PvP).md), [Ход игрока (человек, бот)](Ход игрока (человек, бот).md).
