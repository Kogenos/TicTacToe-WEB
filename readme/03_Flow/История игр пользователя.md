# История игр пользователя

- Эндпоинт: `GET /games/finished` (защищённый).
- Вызывает `GameServiceWithRepo.GetFinishedGamesByUser` → `GameRepository.ListFinishedGamesByUser`.
- SQL-запрос (`list_finished_games_by_user.sql`):
  - Выбирает игры со статусом `finished`, где пользователь является `player_x_id` или `player_o_id`.
  - Сортирует по `create_date DESC`.

**Возвращает:** список `GameDTO` (все поля, включая `create_date`).

**Применение:** клиент может отображать историю партий пользователя.

См. также [Завершение игры и статусы](Завершение игры и статусы.md).
