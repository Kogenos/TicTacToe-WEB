# Singleton

Порождающий паттерн, гарантирующий, что у типа существует только один экземпляр.

## В проекте

- Пул соединений `*pgxpool.Pool` – создаётся один раз в `DatasourceModule` и используется всеми репозиториями.
- `JwtProvider` – также синглтон, создаётся один раз и используется в `AuthService`.

## Как достигается в fx

- fx по умолчанию кеширует результат каждого конструктора – повторные запросы того же типа возвращают тот же экземпляр.
