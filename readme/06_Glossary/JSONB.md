# JSONB

Тип данных PostgreSQL, позволяющий хранить JSON в бинарном формате.

## Использование в проекте

- Поле `game_board` – матрица 3×3.
- Поле `history` – массив объектов ходов.

**Почему JSONB, а не отдельные таблицы?**
- Игровое поле и история – небольшие структуры, которые всегда читаются/пишутся целиком.
- Избегаем сложных JOIN и множества строк.
- Упрощает код и ускоряет операции.

## Обработка в Go

- При сохранении: `json.Marshal` структуры Go в `[]byte`, затем `Exec` с параметром.
- При загрузке: `Scan` в `[]byte`, затем `json.Unmarshal`.
