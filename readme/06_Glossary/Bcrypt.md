# Bcrypt

Алгоритм хеширования паролей, устойчивый к перебору (медленный).

## Использование в проекте

- `bcrypt.GenerateFromPassword([]byte(password), bcrypt.DefaultCost)` – хеширование при регистрации.
- `bcrypt.CompareHashAndPassword([]byte(hash), []byte(password))` – проверка при аутентификации.

## Преимущества

- Автоматическая генерация соли.
- Настраиваемая «стоимость» (количество раундов). `DefaultCost = 10` – хороший баланс.
- Защита от радужных таблиц.
