# datasource

Пакет предназначен для работы с PostgreSQL на низком уровне.

## Файлы

- `data_base.go` – функция `NewPgxPool` (создание пула соединений с проверкой Ping).
- `game_model.go` – структуры `GameModel` и `MoveModel` для хранения в БД (используют `int`, `[3][3]int`, `[]MoveModel`).
- `mapper.go` – преобразование между `domain.Game` и `GameModel` (и обратно).
- (аналогично могли бы быть модели для пользователей, но для простоты используются прямо в репозитории)

**Ключевая идея:** `GameModel` использует JSONB для `game_board` и `history`. Мапперы конвертируют `domain.Cell` ↔ `int` и `domain.Board` ↔ `[3][3]int`.

**Зависимости:** не зависит от других слоёв (только от `domain`).
